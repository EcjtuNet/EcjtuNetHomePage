  • 华东交通大学电气与自动化工程学院是华东交通大学的二级学院，2016年5月学校将原“电气与电子工程学院”更名为“电气与自动化工程学院”。学院前身是机电系，始建于1979年9月，后经历电气工程系、电气与信息工程学院、电气工程学院、电气与电子工程学院几个发展阶段。

    Read More

  • 华东交通大学机电与车辆工程学院是华东交通大学的二级学院，2016年5月，学校将原“机电工程学院”更名为“机电与车辆工程学院”。学院始建于1979年。

    Read More

  • 华东交通大学土木建筑学院创建于1971年，以土木工程学科为主、以交通土建为特色，学院下设4个系、8个本科专业。其中土木工程、给排水科学与工程专业为国家特色专业，交通工程、建筑环境与能源应用工程、建筑学为江西省特色专业，给排水科学与工程专业通过住建部专业评估，是江西省卓越工程师培养计划试点专业。

    Read More

  • 华东交通大学信息工程学院成立于2002年1月，其前身为1979年成立的机电工程系。截至2015年4月，学院现有教职工81人，其中教授13人、副教授24人。有四个本科专业，三个学术型硕士点。

    Read More

  • 华东交通大学软件学院于2002年6月经江西省教育厅批准成立，是华东交通大学的二级学院，属江西省首批示范性软件学院之一。学院现有在校学生近3700人。

    Read More

  • 华东交通大学理学院成立于1979年10月，前身为华东交通大学基础课部。2014年1月，基础科学学院更名为理学院，原基础科学学院于2000年4月由基础课部升格而来，同年开始招收应用化学本科专业学生。2005年学院成功申报了五个硕士点,且于2007年4月开始招收硕士研究生。现任院长陈爱喜。

    Read More

  • 华东交通大学外国语学院成立于2000年3月，在原外语教学部的基础上逐步发展壮大。学院现有翻译、国际贸易、国际旅游三个专业方向，并承担了全校各类各层次学生的公共外语教学任务，课程涉及各类专业的硕士生、本科生（含专升本）、专科生、高职、成教、辅修及双学位专业等。

    Read More

  • 华东交通大学艺术学院成立于2006年3月，从人文学院分离。现有环境设计、产品设计、数字媒体艺术、音乐学、舞蹈表演5个本科专业。其中“艺文”结合的 音乐学+汉语言文学 双专业复合型人才培养模式创新实验区成功入选江西省省级人才培养模式创新实验区，2010年音乐学+汉语言文学专业成为省级特色专业。

    Read More

  • 2004年5月，华东交通大学与南京军区签约培养国防生，2006年3月8日，正式成立国防生学院。根据《国务院、中央军委关于建立依托普通高等教育培养军队干部制度的决定》精神，按照国家教育部、总政治部下达的招生计划，华东交通大学从2005年开始招收南京军区国防生，首批学生200名。

    Read More

  • 华东交通大学经济管理学院（原经济管理系）始建于1983年，学院现有学科涵盖经济学和管理学两大门类，下设经济、管理、会计、统计4个系。现有1个博士点，一级学科硕士点5个，专业硕士点3个。学院现有教职工126人，其中，教授14人（另有兼职教授6人），副教授44人。

    Read More

  • 华东交通大学人文社会科学学院的前身是创建于1979年10月的马列主义教研室。1988年9月在马列主义教研室基础上，并入了德育教研室、大学语文教研室，成立了社会科学部。2000年05月社会科学部撤消，升格为人文社会科学学院，同年9月开始招收法学专业学生。

    Read More

  • 华东交通大学马克思主义学院2011年4月成立，该学院负责组织、管理全校思想政治理论课教学工作，承担马克思主义理论学科点建设和研究生培养任务，开展马克思主义理论的科学研究以及马克思主义理论教学科研师资队伍建设。

    Read More

  • 华东交通大学国际学院是以中外合作办学和外国留学生教育为主要任务的教学单位。学院承担着本科教学改革试验班的教学任务，同时还承担着学校留学生教学安排和管理工作。

    Read More

  • 华东交通大学交通运输与物流学院是华东交通大学下属的二级学院，原名为华东交通大学轨道交通学院，学院成立于2009年，2016年5月，轨道交通学院更名为交通运输与物流学院。

    Read More

  • 华东交通大学材料科学与工程学院是华东交通大学下属的二级学院，2016年5月，华东交通大学通过整合理、机电两个学院和生物材料研究所学科专业资源，新成立“材料科学与工程学院”。
